 정밀 위치 제어란?
기존: “500스텝 이동” 같은 상대 이동
업그레이드: “목표 위치 = 10,000 스텝” 같은 절대 위치 제어
즉,
✅ 현재 위치 저장
✅ 목표 위치 계산
✅ 방향 자동 결정
✅ S-curve로 이동
✅ 리미트 스위치로 위치 보정
✅ Homing 후 절대 좌표계 생성

✅ 3. 전체 통합 코드 (S-curve + 정밀 위치제어 + TB6600)
아래 코드는:
✅ S-curve 가감속
✅ 절대 위치 제어
✅ Homing
✅ 좌/우 리미트
✅ E‑STOP
✅ 포텐셔미터 속도 조절
✅ 시작/정지 버튼
✅ TB6600 제어
까지 모두 포함한 완성형 모션 컨트롤러야.
